Visualizing Data

Data can be visualized without using numbers with a visualization called a word cloud. A word cloud allows you visualize the frequency with which an item appears in a set of data or show the importance of an item in a set of data by using the font size or color of the word to indicate its weight or significance. I used word clouds to show the frequency in which a given category was associated with the movies in the dataset which consisted of movies from the TMDB Top 5000 Movies Dataset as well as elements from the OMDB API.
